Prework Study Guide

HTML

  • HTML stands for Hypertext Markup Language.
  • It is a markup language used to create web pages and other types of digital documents that can be displayed in a web browser.
  • The head element contains information about the webpage.
  • the body element represents the visible content shown to the user.
  • HTML documents are plain text files with a .html file extension.

CSS

  • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
  • CSS is what makes the webpage look cool.
  • If HTML is the Skeleton of the webpage then CSS is the muscle.
  • CSS allows web developers to control the layout, colors, fonts, and other visual aspects of a web page, enabling them to create visually appealing and user-friendly web pages.
